Project 7 - Wearables, Motion, and Phsyiology Stream for Shift Trends

Monitoring one's cognitive load allows us to decide whether someone is in a good mental state before performing crucial tasks. Performing tasks 
that require a high cognitive load for long periods could lead to injury or poor performance. The goal of this project is to track a person's 
cognitive load while they perform their tasks. 

We plan to do this by using wearable devices that can track data and also don’t interfere with the person while they perform their tasks. We 
generally focus on physiological data such as heart rate, skin temperature, and EDA to track cognitive load. Using the live time data that we
collect, while filtering out the noise. Since we are using real time data, one of the goals is to be able to track the data and view trends
while someone is performing their tasks. Since physical and mental strain are often interconnected, we will also factor in these changes when 
assessing cognitive load.

The end goal of this project is to track and identify tasks that cause high cognitive load. By identifying trends that lead to high cognitive 
load, it is possible to set limits or require breaks when a person experiences high cognitive load for long periods. This reduces the risk of 
injuries and errors while performing tasks.
